# Генератор заданий по машинному представлению вещественных чисел

## Описание

Приложение генерирует индивидуальные задания по теме "Машинное представление вещественных чисел" в формате IEEE 754.

## Функциональность

- Генерация случайных вещественных чисел в заданном диапазоне
- Конвертация чисел в машинное представление (32 или 64 бита)
- Обратная конвертация и расчет точности
- Создание файлов в формате Markdown
